Welcome to DOS Commands Tutorial


Hover a command from the list to see its details.



DIR

Directory: Lists the files and directories in the current directory.


Usage:
DIR [drive:][path][filename]

Attributes:
/A[[:]attributes] /B /C /D /L /N /O[[:]sortorder] /P /Q /R /S /T[[:]timefield] /W /X /4

CD

Change Directory: Changes the current directory.


Usage:
CD [drive:][path]

Attributes: None

MD

Make Directory: Creates a new directory.


Usage:
MD [drive:][path]dirname

Attributes: None

RD

Remove Directory: Deletes a directory.


Usage:
RD [drive:][path]dirname

Attributes: None

COPY

Copies one or more files to another location.


Usage:
COPY [drive:][path][filename] [drive:][path][filename]

Attributes: /V /N /Y | /-Y

DEL

Delete: Deletes one or more files.


Usage:
DEL [/P] [/F] [/S] [/Q] [/A[[:]attributes]] names

Attributes: None

REN

Rename: Renames a file or directory.


Usage:
REN [drive:][path][filename] [filename]

Attributes: None

MOVE

Moves one or more files from one directory to another.


Usage:
MOVE [/Y | /-Y] [drive:][path]filename1[,...] destination

Attributes: None

TYPE

Type: Displays the contents of a text file.


Usage:
TYPE [drive:][path]filename

Attributes: None

CLS

Clear: Clears the screen.


Usage:
CLS

Attributes: None

ASSOC

Associates a file extension with a file type (or lists associations).


Usage:
ASSOC [.ext[=[fileType]]]

Attributes: None

ATTRIB

Displays or changes file attributes.


Usage:
ATTRIB [+R | -R] [+A | -A] [+S | -S] [+H | -H] [drive:][path][filename]

Attributes: None

CHKDSK

Checks a disk and displays a status report.


Usage:
CHKDSK [drive:][[path]filename] [/F] [/V]

Attributes: None

DISKPART

Manages disks, partitions, and volumes.


Usage:
DISKPART

Attributes: None

FORMAT

Formats a disk for use with Windows.


Usage:
FORMAT [drive:] [/FS:file-system] [/V:label] [/Q] [/A:size] [/C] [/X]

Attributes: None

LABEL

Creates, changes, or deletes the volume label of a disk.


Usage:
LABEL [drive:][label]

Attributes: None

MD

Creates a new directory.


Usage:
MD [drive:][path]dirname

Attributes: None

MKLINK

Creates a symbolic link to a directory or file.


Usage:
MKLINK [[/D] | [/H] | [/J]] link target

Attributes: None

MORE

Displays output one screen at a time.


Usage:
MORE [/E [/C]] [/P] [/S] [/Tn] [+n] [files]

Attributes: None

NET

Manages network resources.


Usage:
NET [ACCOUNTS | COMPUTER | CONFIG | CONTINUE | FILE | GROUP | HELP | HELPMSG | LOCALGROUP | NAME | PAUSE | PRINT | SEND | SESSION | SHARE | START | STATISTICS | STOP | TIME | USE | USER | VIEW]

Attributes: None

PING

Sends Internet Control Message Protocol (ICMP) Echo Request messages to a specified remote computer.


Usage:
PING [-t] [-a] [-n count] [-l size] [-f] [-i TTL] [-v TOS] [-r count] [-s count] [[-j host-list] | [-k host-list]] [-w timeout] destination-list

Attributes: None

RENAME

Renames a file or files.


Usage:
RENAME [drive:][path][directoryname] oldname newname

Attributes: None

RMDIR

Removes (deletes) a directory.


Usage:
RMDIR [/S] [/Q] [drive:]path

Attributes: None

ROUTE

Manipulates network routing tables.


Usage:
ROUTE [-f] [-p] [-4|-6] command [destination] [MASK netmask] [gateway] [METRIC metric] [IF interface]

Attributes: None

SHUTDOWN

Allows proper local or remote shutdown of machine.


Usage:
SHUTDOWN [/i | /l | /s | /r | /g | /a | /p | /h | /e] [/f]

Attributes: None

SYSTEMINFO

Displays detailed configuration information about a computer and its operating system.


Usage:
SYSTEMINFO [/S system [/U username [/P [password]]]] [/FO format] [/NH]

Attributes: None

TASKLIST

Displays a list of applications, services, and the Process ID (PID) currently running on either a local or a remote computer.


Usage:
TASKLIST [/S system [/U username [/P [password]]]] [/M [module] | /SVC | /V] [/FI filter] [/FO format] [/NH]

Attributes: None

TREE

Graphically displays the directory structure of a drive or path.


Usage:
TREE [drive:][path] [/F] [/A]

Attributes: None

XCOPY

Copies files and directory trees.


Usage:
XCOPY source [destination] [/A | /M] [/D[:date]] [/P] [/S [/E]] [/V] [/W] [/C] [/I] [/Q] [/F] [/L] [/G] [/H] [/R] [/T] [/U] [/K] [/N] [/O] [/X] [/Y] [/-Y] [/Z]

Attributes: None
